🌍 IP Address

Networking Topic

What is an IP Address?

IP Address is a unique address given to a device connected to a network. It helps devices communicate with each other.

Types of IP Address

  • IPv4
  • IPv6
  • Public IP
  • Private IP

Example

192.168.1.10

Why IP Address is Important?

  • Identifies devices
  • Allows communication
  • Helps routing data
